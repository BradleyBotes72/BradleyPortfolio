# Dashboard Images

Place your Power BI dashboard screenshots here.

## Recommended Format

- **File Type**: PNG or JPG
- **Dimensions**: 1920x1080 or 1600x900 (16:9 aspect ratio)
- **File Size**: Under 500KB for optimal loading
- **Naming**: dashboard1.png, dashboard2.png, etc.

## How to Capture Screenshots

### From Power BI Service:
1. Open your report in Power BI Service
2. Use Windows Snipping Tool (Win + Shift + S)
3. Capture the full dashboard area
4. Save as PNG in this directory

### From Power BI Desktop:
1. Open your report
2. View > Full Screen
3. Capture screenshot
4. Save in this directory

## Optimization Tips

- Use tools like TinyPNG or ImageOptim to compress images
- Ensure images are clear and readable
- Consider adding a subtle border or shadow for better presentation
